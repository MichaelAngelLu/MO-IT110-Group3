1. Open Command Prompt (Windows) or Terminal (macOS/Linux): Open CMD or Terminal on your computer.

2. Navigate to the Directory Containing the JAR File: Use the cd command to navigate to the directory containing the JAR file you want to run.

Example:
cd path/to/your/directory

3. Run the JAR File: Once you're in the directory containing the JAR file, you can run it using the java command followed by the name of the JAR file.

Example:
java -jar Group3-MS2.jar

Note: Replace yourJarFile.jar with the actual name of your JAR file.

4. Press Enter: After typing the command, press Enter to execute it.
